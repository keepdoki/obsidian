## 单利复利计算公式
### 1. 单利：$A=P \times(1+nr)$
### 2. 复利：$A=P \times (1+r)^n$ 
> A是未来值; P是本金; r是年利率; n是投资期限;

> 例如存款100000，一年利率5%，存6年
1. 单利：$100000 \times (1 + 0.05 \times 6) = 130000$
2. 复利：$100000 \times (1+0.05)^6 = 134009.5640625$
